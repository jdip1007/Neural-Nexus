## Add Concept

To add a new concept, create a markdown file in the `concepts/` directory following the [schema guidelines](../SCHEMA.md#concepts).

## Concept Characteristics

- **Foundational**: Provide basic understanding and framework
- **Comprehensive**: Cover theoretical and practical aspects
- **Interconnected**: Link to related concepts and entities
- **Evidence-based**: Supported by research and data

## Quality Standards

- Must include proper citations to sources
- Should balance theory with practical applications
- Include examples and case studies where appropriate
- Provide clear conceptual frameworks and definitions
