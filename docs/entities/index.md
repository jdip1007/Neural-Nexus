## Add Entity

To add a new entity, create a markdown file in the `entities/` directory following the [schema guidelines](../SCHEMA.md#entities).

## Entity Categories

- **People**: Researchers, authors, contributors, historical figures
- **Organizations**: Institutions, companies, labs, government agencies
- **Projects**: Research projects, initiatives, programs
- **Tools**: Software, equipment, methodologies

## Quality Standards

- Must include verifiable information and sources
- Should provide context and significance of the entity
- Include relevant dates, locations, and relationships
- Maintain consistent formatting and structure
